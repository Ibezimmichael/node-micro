# node-micro

Repo for basic common tooling needed for any javascript project

## Requirements

- Node.js v 16.18.x or higher
- github access

## installation

```
git clone https://github.com/node-micro.git
cd node-micro
npm install

npm run start
```
## setup
```
cp .env.examples .env
npm run dev or npm start:dev

```

## deploy
```
npm run test
npm run build
npm run deploy
```
